    function solution(n) {
        let answer = 0
        for(let i = 1; i <= n; i++) {
            if(n % i === 0) {
                answer = answer + i
            }
        }
        return answer
    }

++++++++++++++++++++++

    약수의 합을 구하는 문제였다.

    우선 answer 변수에 0을 할당해주고,
    for문을 실행시켜서 약수를 찾아주고,
    그 약수들을 하나하나씩 answer에 더해주는 방식으로 풀어봤다.
    약수를 구하는 방법만 알고 있다면 쉽게 풀 수 있는 문제인 것 같다.
